💬 Mooas AICC (음성봇/챗봇 플랫폼)

기간 2024.01–2024.12 · 역할 AICC 챗봇·음성봇 & RAG 서버 메인 · 기술 Flask, Oracle, Redis, Kore.ai

Problem
  • Rate Limit/지연 증가, 문서 일치도·정확도 편차
Solution
  • 임베딩·벡터검색 기반 RAG 파이프라인 구축(FAISS/NPY)
  • Redis 60s 캐시로 동일 질의 비용/지연 감소
  • 모델 티어 자동 전환으로 쿼터 대응
  • Gunicorn 워커/스레드 튜닝(+Nginx)으로 동시성 확장
Impact
  • 동일 질의 응답 체감 지연 감소
  • LLM 호출·비용 절감(범위 표기)
  • 서비스 안정화 및 운영 편의성 향상
Artifacts
  • Tech Docs (공개 요약)
  • 아키텍처 다이어그램
  • 서비스 소개

🎟 CREPAS (NFT 마켓 & 티켓) 서비스 종료

기간 2022.06–2024.01 · 역할 관리자 풀스택 & 블록체인 메인 · 기술 React, Laravel, Truffle, Polygon, ERC-721/1155, AWS

Problem
  • 이벤트/티켓 등 대량 발행 시 타임아웃·중복민팅 등 안정성 이슈
  • ERC-721/1155 혼합 발행과 수수료/재발행 관리 필요
Solution
  • Truffle + HDWalletProvider로 Polygon(Mumbai/Mainnet) 배포 파이프라인 구축
  • 커스텀 ERC-721/1155 설계(권한·URI·이벤트) 및 Polygonscan 검증
  • 메타데이터/이미지 S3 또는 IPFS 저장 → URI로 민팅
  • 민팅 API + 발행 큐(대기열/배치)로 피크 트래픽 제어
Impact
  • 피크 구간 타임아웃/중복발행 대폭 감소
  • 운영 자동화로 발행 리드타임 단축 및 오류율 감소

※ 수치는 내부 정책상 범위로 표기
